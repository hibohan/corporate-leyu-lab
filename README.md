# corporate-leyu-lab

本仓库用于归档和发布一系列独立 HTML 页面。这些页面各自独立，不针对特定域名或具体网站，仅作为静态资源存档与展示用途。

## 项目简介

corporate-leyu-lab 是一个以 HTML 页面为核心的文件仓库。所有页面均为独立文件，无后端依赖，可直接在浏览器中打开或通过静态托管服务访问。仓库内容可能包含信息展示、交互原型、说明文档等类型页面，具体用途由文件本身定义。

## 目录说明

```
corporate-leyu-lab/
├── index.html          # 仓库入口页面（如有）
├── pages/              # 归档页面存放目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

目录结构可能随内容更新调整，请以实际仓库文件为准。

## 页面归档说明

- 每个 HTML 页面均为独立完成，不依赖外部动态接口。
- 页面内容不指向任何特定网站或服务，仅供存档与参考。
- 归档文件可能包含演示、测试或信息展示性质的内容。
- 部分页面可能包含交互元素（如 JavaScript），请在本地或静态环境下运行。
